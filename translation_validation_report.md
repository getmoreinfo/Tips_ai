## 문제 유형 분류

### 1. 브랜드명/고유명사 번역 문제 (10/10건)
- **심각도**: HIGH (5건), MEDIUM (5건)
- **원인**: TranslateGemma가 고유명사를 일반 단어로 해석하여 번역
- **해결 방안**: 
  - 번역 전 고유명사 마킹/보존
  - 번역 후 고유명사 복원
  - 프롬프트에 "브랜드명은 번역하지 말 것" 지시 추가

### 2. 키워드 오번역 (2건)
- 고구마 → khoai tây (감자) ❌ → khoai lang (고구마) ✅
- 기저귀 → 누락 ❌ → tã/bỉm ✅

### 3. 숫자/정보 누락 (1건)
- "1년" → 누락 ❌ → "1 năm" ✅

## 권장 해결 방안

### 즉시 적용 가능한 방법

1. **고유명사 사전 구축**
   ```python
   BRAND_NAMES = {
       '그린키즈': 'Green Kids',
       '반달': 'Bandal',
       '어스본': 'Usborne',
       '지학사': 'Jihak Publishing',
       '아가짱': 'Agachan',
       '폴레드': 'Poled',
       '에르고베이비': 'Ergobaby',
       '새들베이비': 'SaddleBaby',
       '헬로미니미': 'Hello Mini'
   }
   ```

2. **번역 전/후 처리**
   - 번역 전: 고유명사를 플레이스홀더로 치환
   - 번역 후: 플레이스홀더를 원본 또는 표준명으로 복원

3. **프롬프트 개선**
   - "다음 텍스트에서 브랜드명과 고유명사는 번역하지 말고 그대로 보존하세요" 지시 추가

### 장기 개선 방안

1. **Fine-tuning**: 고유명사 보존을 학습한 커스텀 모델
2. **후처리 규칙**: 정규표현식 기반 고유명사 복원
3. **품질 검증**: 번역 후 자동 검증 스크립트
