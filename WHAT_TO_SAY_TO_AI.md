# 다른 컴퓨터에서 Cursor AI에게 할 말 가이드

## 다른 컴퓨터에서 Git Pull 완료 후

### 1단계: 패키지 설치 확인 및 실행

**Cursor에서 AI에게:**

```
이 프로젝트에 필요한 Python 패키지를 설치해야 합니다. 
setup_other_computer.ps1 스크립트를 실행하는 방법을 알려주세요.
```

또는 직접 실행:
```bash
.\setup_other_computer.ps1
```

---

### 2단계: .env 파일 생성

**Cursor에서 AI에게:**

```
.env 파일이 없는데 create_env_file.py를 실행해서 생성해야 합니다.
실행 방법을 알려주세요.
```

또는 직접 실행:
```bash
python create_env_file.py
```

**그 후 AI에게:**
```
.env 파일이 생성되었습니다. 이제 실제 데이터베이스 연결 정보로 수정해야 합니다.
어떤 값들을 변경해야 하는지 알려주세요.
```

---

### 3단계: 빠른 테스트 실행 (선택사항)

**Cursor에서 AI에게:**

```
test_training_quick.py 파일이 있는데 이걸 실행하면 학습이 정상 작동하는지 확인할 수 있다고 했습니다.
실행 방법을 알려주세요.
```

또는 직접 실행:
```bash
python test_training_quick.py
```

**정상 작동 확인 후 AI에게:**
```
테스트가 성공했습니다. 이제 이 컴퓨터는 멀티 노드 분산 학습을 위한 준비가 완료되었나요?
다음 단계가 무엇인지 알려주세요.
```

---

### 4단계: Accelerate 설정 (멀티 노드용)

**Cursor에서 AI에게:**

```
이 컴퓨터는 3대의 컴퓨터 중 하나입니다. 멀티 노드 분산 학습을 위해 Accelerate를 설정해야 합니다.
accelerate config 명령을 실행하는 방법과 어떤 값들을 설정해야 하는지 알려주세요.
```

**또는 더 구체적으로:**

```
3대의 컴퓨터로 분산 학습을 하려고 합니다.
- 컴퓨터 1 (메인): IP 192.168.1.100, rank 0
- 컴퓨터 2 (이 컴퓨터): IP 192.168.1.101, rank 1  
- 컴퓨터 3: IP 192.168.1.102, rank 2

이 컴퓨터(컴퓨터 2)에서 accelerate config를 어떻게 설정해야 하는지 알려주세요.
```

---

## 간단한 버전: 한 번에 모든 것을

**Cursor에서 AI에게 이렇게 말하면:**

```
안녕하세요. 이 컴퓨터는 AI 학습 프로젝트를 위한 두 번째 컴퓨터입니다.
Git pull을 완료했고, 이제 환경 설정을 완료해야 합니다.

다음 단계를 순서대로 알려주세요:
1. 필요한 Python 패키지 설치
2. .env 파일 생성 및 설정
3. 학습 테스트 실행
4. Accelerate 멀티 노드 설정 (이 컴퓨터는 rank 1입니다)

각 단계를 실행하는 방법과 주의사항을 알려주세요.
```

## 체크리스트: AI에게 확인할 것들

### 패키지 설치 후:
- [ ] "필요한 패키지가 모두 설치되었는지 확인해주세요."
- [ ] "GPU가 정상적으로 인식되는지 확인해주세요."
- [ ] `nvidia-smi` 실행 결과 확인

### .env 파일 설정 후:
- [ ] ".env 파일에 올바른 값이 설정되었는지 확인해주세요."
- [ ] "데이터베이스 연결 테스트를 실행하는 방법을 알려주세요."

### 테스트 실행 후:
- [ ] "테스트가 성공했다면 다음 단계가 무엇인가요?"
- [ ] "이제 멀티 노드 학습을 시작할 준비가 되었나요?"

### Accelerate 설정 후:
- [ ] "Accelerate 설정이 올바른지 확인해주세요."
- [ ] "다른 컴퓨터들과 연결하기 전에 확인해야 할 사항이 있나요?"

## 문제 발생 시 AI에게

### 오류 발생 시:
```
[오류 메시지] 라는 오류가 발생했습니다. 
이 문제를 해결하는 방법을 알려주세요.
```

### 확인이 필요할 때:
```
이 설정이 올바른지 확인해주세요.
[설정 내용 또는 파일 경로]
```

### 다음 단계 확인:
```
현재까지 완료한 것:
1. Git pull 완료
2. 패키지 설치 완료
3. .env 파일 생성 완료

다음에 해야 할 일이 무엇인가요?
```

## 가장 간단한 방법: 프로젝트 상황 설명

**Cursor에서 AI에게 이렇게 말하면 됩니다:**

```
안녕하세요. 

이 컴퓨터는 AI 학습 프로젝트를 위한 두 번째(또는 세 번째) 컴퓨터입니다.
Git pull을 완료했고, 이제 환경 설정을 시작하려고 합니다.

현재 프로젝트 상황:
- 3대의 컴퓨터로 분산 학습을 진행하려고 합니다
- 메인 컴퓨터에서 모든 설정을 테스트하고 Git에 커밋했습니다
- 이 컴퓨터에서 같은 환경을 구성해야 합니다

필요한 작업:
1. Python 패키지 설치 (setup_other_computer.ps1 실행)
2. .env 파일 생성 및 설정
3. 학습이 정상 작동하는지 테스트
4. Accelerate 멀티 노드 설정

각 단계를 순서대로 안내해주시고, 실행할 명령어를 알려주세요.
```

## 실제 사용 예시

### 예시 1: 간단하게
```
다른 컴퓨터에서 Git pull 완료했습니다. 
이제 환경 설정을 위해 setup_other_computer.ps1를 실행하면 되나요?
```

### 예시 2: 자세하게
```
안녕하세요. 

이 컴퓨터는 3대의 분산 학습 컴퓨터 중 두 번째 컴퓨터입니다.
Git pull을 완료했고, 프로젝트 파일들이 모두 받아졌습니다.

다음 단계를 진행하려고 합니다:
1. setup_other_computer.ps1 실행하여 패키지 설치
2. create_env_file.py 실행하여 .env 파일 생성
3. test_training_quick.py 실행하여 학습 테스트

각 단계를 어떻게 실행하면 되는지, 그리고 주의사항이 있다면 알려주세요.
```

### 예시 3: 문제 해결
```
test_training_quick.py를 실행했는데 
[오류 메시지] 라는 오류가 발생했습니다.
이 문제를 어떻게 해결해야 하나요?
```
